# 상담문의 → 구글 시트 + 자동 응답메일 연동 가이드

상담폼으로 들어온 문의를 **구글 스프레드시트에 자동 기록**하고, 관리자 알림 메일과 방문자 자동
응답메일을 보내는 설정입니다. 약 10분 소요되며, 비용은 무료입니다.

## 1단계. 구글 시트 만들기

1. https://sheets.google.com 접속 → **새 스프레드시트** 만들기
2. 이름을 예: `케이앤에스 상담문의 접수`로 변경

## 2단계. Apps Script 붙여넣기

1. 시트 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
2. 편집기에 기본으로 있는 코드를 모두 지우고, 이 폴더의 **`google-apps-script.gs`** 파일 내용을
   전체 복사해 붙여넣기
3. 상단의 `ADMIN_EMAIL` 값이 알림받을 메일 주소(기본: kglobal4756@gmail.com)인지 확인
4. 💾 저장 (프로젝트 이름은 아무거나, 예: `상담문의수신`)

## 3단계. 웹앱으로 배포

1. 오른쪽 위 **배포 → 새 배포** 클릭
2. 톱니바퀴(⚙) → **웹 앱** 선택
3. 설정:
   - 설명: `상담문의 수신`
   - 다음 사용자 인증 정보로 실행: **나**
   - 액세스 권한이 있는 사용자: **모든 사용자** ← 중요!
4. **배포** 클릭 → 구글 계정 권한 승인 (처음 한 번, "안전하지 않은 페이지" 경고가 나오면
   `고급 → 프로젝트로 이동`을 눌러 승인)
5. 발급된 **웹 앱 URL**(https://script.google.com/macros/s/…/exec)을 복사

## 4단계. 홈페이지에 URL 등록

프로젝트 루트의 `.env` 파일을 열고:

```
SHEETS_WEBHOOK_URL="복사한 웹 앱 URL"
```

저장 후 서버를 재시작하면 완료입니다.

## 동작 확인

홈페이지 상담문의 폼으로 테스트 문의를 보내면:

1. 구글 시트 `상담문의` 탭에 한 줄 추가
2. 관리자 메일로 `[홈페이지 상담문의] 이름/분야` 알림 도착
3. 문의 시 이메일을 입력했다면 그 주소로 접수확인 메일 도착
4. 관리자 콘솔(/admin/inquiries)에도 동일하게 기록 (시트 연동과 무관하게 항상 저장)

## 자주 묻는 질문

- **메일이 스팸함으로 가요** → 자동 발송 메일은 구글 계정 명의로 발송됩니다. 처음 몇 건은
  스팸함을 확인하고 "스팸 아님" 처리해 주세요.
- **하루 발송 한도** → 일반 구글 계정은 하루 약 100건까지 메일 발송이 가능합니다. 행정사무소
  상담 규모에는 충분합니다.
- **스크립트를 수정했어요** → 수정 후에는 **배포 → 배포 관리 → 수정(연필) → 새 버전 → 배포**를
  해야 반영됩니다.
- **개인정보 보호** → 시트에는 문의자의 이름·연락처가 저장되므로, 시트 공유 설정을 "나만 보기"로
  유지하세요. 보유기간(1년) 경과 건은 주기적으로 삭제하는 것을 권장합니다.
